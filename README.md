# AWS DevOps Lab

Primer laboratorio práctico de Cloud/DevOps.

## Qué he hecho

- Creación de una instancia EC2 en AWS
- Instalación de Ubuntu Server
- Conexión al servidor
- Instalación de Nginx
- Despliegue de una web básica
- Revisión de logs de Nginx
- Uso básico de Bash scripting
- Automatización con Cron
- Control de versiones con Git
- Subida del proyecto a GitHub

## Tecnologías usadas

- AWS EC2
- Ubuntu Server
- Nginx
- Bash
- Cron
- Git
- GitHub

## Objetivo

Aprender los fundamentos de administración Linux, Cloud y DevOps desde cero.
